# QR Code Generator

A simple and interactive project to generate QR codes from any text or URL using HTML, CSS, and JavaScript.

## Live Demo
Try it live here: [QR Code Generator](https://asma-ehsan.github.io/QR-Code-Generator/)

## Features
- Generate QR codes instantly from any input text or URL.
- Input validation with a shake animation for empty fields.
- Smooth transition when displaying the generated QR code.
- Modern and clean UI for better user experience.
- Built using vanilla JavaScript and free public QR code API.

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Enter any text or URL in the input field.
4. Click the **Generate QR Code** button.
5. The QR code will appear below the input field.
